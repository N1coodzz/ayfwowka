# Polymarket Replit Test Bot

Все тестовые данные уже вставлены в `main.py`.

Нажми `Run`.

После запуска в Telegram должно прийти:

```text
✅ Replit bot запущен.
```

Команды:

```text
/start
/ping_polymarket
/test_alert
/stats
/bets
/export
```

Важно: после теста перевыпусти Telegram bot token в BotFather.
