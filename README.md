# Polymarket Replit Inline Auth Bot

Тестовая версия для Replit.

## Уже вставлено в код

- Telegram bot token
- Polymarket wallet
- Пароль входа: `OOORASSVET`

После теста обязательно перевыпусти token в BotFather.

## Запуск

1. Создай новый Replit project.
2. Загрузи эти файлы:
   - `main.py`
   - `requirements.txt`
   - `.replit`
   - `README.md`
3. Нажми `Run`.

## Как войти

В Telegram открой бота и напиши:

```text
/start
```

Потом введи пароль:

```text
OOORASSVET
```

После этого всё управление будет через inline-кнопки.

## Что умеет

- Inline-меню без команд.
- Доступ только после пароля.
- Каждый пользователь ведёт свою статистику.
- У каждого пользователя свой Excel-файл.
- Новые сделки игрока рассылаются всем авторизованным пользователям.
- В алерте показывается:
  - цена сделки в центах и decimal;
  - размер конкретной сделки в USDC;
  - shares конкретной сделки;
  - ссылка на рынок;
  - tx hash;
  - текущая позиция игрока по outcome, если positions API вернул данные.
- Кнопки:
  - Повторить;
  - Открыть рынок;
  - Последние 10 сделок игрока;
  - Статистика;
  - Мои ставки;
  - Открытые;
  - Excel;
  - Тест алерт.
